# yanpgxue 的作品集

基于 GitHub Pages 的免费个人作品展示站。

## 文件说明
- `index.html` —— 网页内容
- `style.css`  —— 深色风格样式（已做手机适配）
- `assets/snow.png` —— Snow Player 软件界面截图

## 本地预览
直接双击 `index.html` 用浏览器打开即可看效果。

## 免费上线（GitHub Pages）
1. 新建仓库，名字必须是：`yanpgxue.github.io`（选 Public）
2. 把本目录所有文件（含 `assets/`）上传到仓库
3. 仓库 Settings → Pages → Branch 选 `main` → Save
4. 访问 https://yanpgxue.github.io

## 怎么改
- 改文字：编辑 `index.html` 里的名字、简介、链接
- 加作品：复制一个 `<div class="card">...</div>` 块
- 换截图：替换 `assets/snow.png`，或在 `index.html` 改 `<img src="...">`
