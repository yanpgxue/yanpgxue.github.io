创作过程

以 Snow Player 为例，从想法到成品的 5 个步骤 👇

1. 灵感来源

市面上的音乐 App 各自为政，想听不同平台的歌得来回切换。于是想自己做一个能聚合多平台音乐的桌面播放器。

2. 技术选型

用 Python 写逻辑，界面用 PySide6（Qt 的 Python 绑定），跨平台、组件丰富，适合做桌面应用。

3. 多源搜索

接入酷狗、网易云、酷我、QQ音乐的非官方接口，输入歌名就能一次搜出四个平台的结果。

4. 播放与下载

双击歌曲名即时试听，支持换背景图；歌词实时同步显示，听歌体验更完整。

5. 打包发布

用 PyInstaller 把脚本打成 exe，再用 Inno Setup 做成可安装 / 卸载的安装包，方便分享给别人。
